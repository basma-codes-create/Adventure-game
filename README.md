# Adventure Game (Text-Based Python Game)

A text-based adventure game built in Python where the player explores different environments, makes decisions, solves puzzles, and tries to survive with limited lives.

---

## Game Description

The player wakes up in a mysterious forest and progresses through three main levels:

- The Mysterious Forest  
- Ancient Ruins  
- The Wizard’s Tower  

Each level contains choices, random events, traps, and puzzles that affect the player’s score and remaining lives.

---

## Features

- Lives system (3 lives)
- Score system based on decisions
- Random events for replayability
- Multiple choices per level
- Riddle-based final challenge
- Ending system based on final score
- Option to replay the game

---

## How to Play

During the game, the player is prompted to choose between options such as:

Enter (1-3):

The player types the number of their choice and presses Enter.

Each choice affects:
- Score increase or decrease
- Loss of lives in case of traps
- Progress through levels
- Final outcome

---

## Game Levels

### Level 1: The Mysterious Forest
- Explore different paths
- Encounter random events
- Learn basic survival mechanics

### Level 2: Ancient Ruins
- Navigate through hidden paths
- Avoid traps
- Find an ancient treasure

### Level 3: The Wizard’s Tower
- Solve riddles
- Limited attempts per question
- Final challenge to win the game

---

## Endings

The final result depends on the player’s score:

- 70 or higher: Legendary Explorer Ending  
- 40 to 69: Good Survival Ending  
- Below 40: Bad Ending  

---

## Requirements

- Python 3.x  
- No external libraries required (only time and random modules)
